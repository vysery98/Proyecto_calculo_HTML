Prioridad de los Operadores


Los problemas matemáticos son realizados considerando los operadores y se los realiza en el siguiente orden indicado:

Prioridad Operador Prioridad Operador
1 {...} Llaves 6 + Suma y - Resta
2 [...] Corchetes 7 Operadores de comparación <, >, ≤, ≥, etc.
3 (...) Paréntesis 8 Operadores de igualdad = y ≠
4 xn Potencias y n√x Raíces 9 Operadores lógicos if, and, or, else, then, until, etc.
5 + Suma y - Resta

Ejemplo:

- 2 { - 3 [ - 1 + (- 2) ] 2 } - 1 [ 2 + (- 5) ] 3

  1. - 2 [ - 3 ( - 1 - 2 ) 2 ] - 1 ( 2 - 5 ) 3
    Se realizó las operaciones dentro del paréntesis, en este caso el producto de dos números aplicando la ley de signos: + ( - 2 ) = - 2; + ( - 5 ) = - 5

  2. - 2 [ - 3 ( - 3 ) 2 ] - 1 ( - 3 ) 3
    Se realizó operaciones artméticas dentro del paréntesis: ( - 1 - 2 ) = - 3; ( 2 - 5 ) = - 3

  3. - 2 [ - 3 ( + 9 ) ] - 1 ( - 27 )
    Se aplicó la potenciación asignada a cada paréntesis: ( - 3 ) 2 = + 9; ( - 3 ) 3 = - 27

  4. - 2 ( - 27 ) + 27
    Se realizó el producto a cada número entre paréntesis, tomando en cuenta la ley de signos y eliminando los paréntesis: - 3 ( + 9 ) = - 27; - 1 ( - 27 ) = + 27

  5. + 54 + 27
    Se realizó el producto del número entre corchetes, tomando en cuenta la ley de signos y eliminando los paréntesis: - 2 ( - 27 ) = + 54

  6. + 81
    Se realizó la suma aritmética: + 54 + 27 = + 81
