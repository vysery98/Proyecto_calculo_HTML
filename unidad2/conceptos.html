FUNCIÓN


Una función es una relación entre dos conjuntos A y B, donde a cada elemento de A le corresponde un único elemento de B.

El conjunto de todos los elementos de B relacionados con algún elemento de A se denomina rango, o conjunto imagen y a cada elemento del conjunto B le denominamos imágen de algún elemento del conjunto A

Notación usual: ƒ: A → B.
Dónde: y = ƒ(x)

– x es la variable independiente

– y es la variable dependiente
Dominio: Es el conjunto de todos los valores que puede tomar la variable independiente "x".

Rango: Es el conjunto de todos los valores que puede tomar una función, dependiendo de los valores de "x" (variable dependiente).
