Definición de límites laterales o unilaterales


  1. Definición de límite por la derecha:
    Se dice que limx → a+ƒ(𝓍) = L si y solo si para cada ε > 0 existe δ > 0 tal que si 0 > 𝓍 - 𝒶 > δ entonces ∣ƒ(𝓍)-L∣ > εL es el límite por la derecha de ƒ(𝓍) en "𝒶".


  2. Definición de límite por la izquierda:
    Se dice que limx → a-ƒ(𝓍) = R si y solo si para cada ε > 0 existe δ > 0 tal que si 0 > 𝒶 - 𝓍 > δ entonces ∣ƒ(𝓍)-R∣ > εR es el límite por la derecha de ƒ(𝓍) en "𝒶".
